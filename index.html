Bienvenue chez Getobz

Bienvenue sur mon site internet, je me présente:

Je m'appelle Bryan, j'ai 23 ans, je vis dans la région du Grand-Est et j'apprends le développement web plus précisement le front-end.


Le fait de pouvoir créer un site internet m'a toujours passionné et même si l'apprentissage peut parfois être difficile, lorsque l'on obtiens le résultat attendu, on est toujours fière du temps et du travail que l'on a investi.


J'ai commencé mon apprentissage grâce à l'aide de la communauté Foutu Code qui m'apporte l'aide appropriée en toute sympathie et en me fournissant la/les documentations nécessaires.


Mon object final est de savoir créer un site internet complet avec les langages suivants:


  • HTML
  • CSS
  • JavaScript
  • PHP
